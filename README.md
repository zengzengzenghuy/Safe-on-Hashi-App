# Safe on Hashi -UI

## Running the app

```
yarn install
yarn start
```

Then:

- Go to [Safe app](https://safe.global/)
- Go to Apps -> Manage Apps -> Add Custom App
- Paste your localhost URL, default is http://localhost:3000/
- You should see Safe Test App as a new app
